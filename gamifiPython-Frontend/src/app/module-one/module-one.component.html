Hablando con una maquina

Entender como una maquina nos acaba entendiendo puede llegar a ser algo confuso. Pero al entender el lenguaje que habla una computadora esto se vuelve muy simple!



Como en cualquier lenguaje humano, este sigue sus propias reglas y sintaxis para escribirse de manera correcta! Una de las mas importantes es la indentacion.

La identacion es el espacion que existe entre el codigo y el inicio de la linea. Usualmente esto es una buena practica en la programacion para organizar el codigo pero en Python la indentacion es completamente necesaria
Veamos unos ejemplos

if 2 > 1:
print("Uno es mayor que dos.")

if 2 > 1:
  print("Uno es mayor que dos, pero indentado!.")
El texto y las ordenes son exactamente iguales. Pero Python lee estos dos ejemplos de maneras completamente diferentes. El lenguaje, al intentar comprender lo que le estamos diciendo, necesita las indentaciones para entender que una parte del codigo depende de otra anterior.

Entre textos y numeros

Las variables se pueden entender como cajas, cada una guardando informacion bajo un nombre unico.


Y en codigo se veria asi: 

x = 4
y = 2.5
q = "hello"
z = "a"
Estas variables son diferentes, pero todas tienen un sentido para Python.
x es un int el cual guarda numeros enteros.
y es un float que tambien guarda numeros! Pero permite los decimales. Estos tienen que estar separados con un punto.!
q es un str o mejor conocido como string. Aun que este paresca solo texto, en realida es un hilo de caracteres.
Y finalmente z es un str aun que solo guarde un solo caracter de teclado. En otros lenguajes z seria de tipo char, pero Python no utiliza este tipo.
Estas variables son usadas en diferentes operaciones respectivas para sus tipos. Los numeros en operaciones aritmeticas y los textos en operaciones de edicion y escritura.

Respuestas logicas

Ademas, Python toma uso de un tipo de datos mas abstracto, el booleano o bool el cual es una respuesta logica!
Esta es una de las primeras variables en todos los lenguajes de programacion, y representa el estado de un solo valor verdadero o falso.


En el codigo seria: 

verdad = true
falso = false
Y asi de simple es este tipo de variable. Los valoes true o false son usados para operaciones logicas dentro de el funcionamiento de Python. Estas las veremos luego en el siguiente segmento.

Operadores y herramientas

Python contiene una gran cantidad de operadores, diferentes herramientas que el lenguaje nos brinda.

Operadores Aritmeticos

Los operadoes aritmeticos son todos los operadores que se puedan usar con valores numericos.
Python brinda todos los operadores aritmeticos tradicionales y adicional a esto se presentan operadores no tan conocidos que son utiles en temas de programacion.

+ | Suma | 5 + 2 = 7
- | Resta | 5 - 2 = 3
* | Multiplicacion | 5 * 2 = 10
/ | Division | 5 / 2 = 2.5
% | Modulo o Resto | 5 % 2 = 5
** | Exponente | 5 ** 2 = 25
// | Division baja | 5 // 2 = 2

Operadoes Lógicos

Los operadores logicos son aquellos usados principalmente con las variables booleanas.
Entre estos existen las regulares expresiones de comparacion de valores numericos al igual que los conectores logicos and, or y not.

if (booleanUno and booleanDos)
  print("booleanUno y booleanDos son ambos valores booleanos verdaderos!")

if (not booleanTres)
  print("booleanTres no es verdadero!")

if (2 <= 10)
  print("Dos es menor o igual que diez!")
